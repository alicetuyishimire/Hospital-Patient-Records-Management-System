Project Title
 
Hospital Patient Records Management System
 
Project Overview 

The Hospital Patient Records Management System is a database project designed to 
manage patient information, medical history, and appointments in a hospital environment.
 
Many hospitals still rely on paper-based or poorly organized systems, which leads to errors, 
data loss, and slow access to patient records. 

This project focuses on designing and implementing a relational database using MySQL to 
securely store and manage patient records in an efficient and structured way.


 SQL Implementation 

Using MySQL, write SQL queries to: 

1. Create the database. 
2. Create all required tables with proper constraints. 
3. Insert the provided sample data. 
4. Retrieve patient medical history. 
5. List patients who visited the hospital on a specific date. 
6. Update patient information. 
7. Delete a patient record. 
8. Use JOINs to generate patient reports.

ANSUERS OF ALL QUESTIONS.

1. Create  and use the database. 

use hospital_db ;

2. Create all required tables with proper constraints.

-- patients table creation.

create table patients(
 patient_id int not null ,
 first_name varchar(50),
 last_name varchar(50),
 gender varchar(10),
 date_of_birth date ,
 phone varchar(15),
 primary key (patient_id));

-- doctors table creation.


 create  table doctors (
 doctor_id int not null,
 doctor_name varchar(100),
 specialization varchar(100),
 primary key (doctor_id));

-- appointments table creation.


 create table appointments (
 appointment_id int not null,
 patient_id int not null,
 doctor_id int not null,
 appointment_date date,
 diagnosis varchar (200),
 primary key (appointment_id));

5. List patients who visited the hospital on a specific date.

select patients.patient_id, patients.first_name, patients.last_name, patients.gender, patients.date_of_birth, patients.phone,
appointments.appointment_date, appointments.diagnosis from appointments inner join patients on appointments.patient_id=patients.patient_id;
